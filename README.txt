README
------

Project: An STE-based methodology for constructing Bayesian Belief Networks as applied to the prediction of Economic Indicators

Class: CS 8750 Artificial Intelligent II

Date: Spring 2015

Instructor: Dr. Yi Shange

Team:
  Fernando Torre Mora   : fthc8@mail.missouri.edu
  Chanmann Lim          : cl9p8@mail.missouri.edu
  Adil Al-azzawi        : aaadn5@mail.missouri.edu

Repository: https://github.com/FernandoSapient/AI2-final

Programming language: Matlab

Library dependencies:
  STEYXY.m : http://www.mathworks.com/matlabcentral/fileexchange/48070-monte-carlo-example-of-a-multi-commodity-spot-and-forward-curves-simulator/content/STEYXY.m

Running the code:
1. Import the dataset and scaling
  > run dataset_importing.m

2. Calculate STE dependencies table
  > run All_STE_tables.m

3. Compute the accuracy of the belief network 1
  > run bn1.m
